# cpp_obfus
Инструмент для преобразования С++ кода в трудночитаемую форму с сохранение работоспособности

# Описание:
cpp_obfus нацелен на усложнение работы с исходным кодом, он вносит косметические изменения в исходный код, а именно:
  1. Удаляет многострочные и однострочные комментарии
  2. По возможности удаляет переводы строк и пробельные символы по краям строк
  3. По возможности перегружает функции
  4. Переименовывает все переменные и функции на сгенерированный набор символов
  5. Сохраняет директивы препроцессора и строковые литерали

# Описание функций:
Основной функцией является obfuscate, которая включает в себя:
  1. Резервирование директив препроцессора и строковых литералов
  2. Удаление комментариев
  3. Перегрузка функций
  4. Переименование функций и переменных
  5. Удаление пустого пространства
  6. Восстановление директив препроцессора и строковых литералов
В результате получается строка, представляющая собой преобразованный код C++, который при запуске будет работать идентично исходному варианту

# Пример использования:
```python3
import cpp_obfus

obfuscate('main.cpp', 'obfuscated_main.cpp)
```

### Требования
- Python 3.6+
- LLVM/Clang (для libclang.dll)

### Установка LLVM
1. Скачайте LLVM с [официального сайта](https://releases.llvm.org/download.html)
2. Установите, выбрав "Add LLVM to system PATH"

### Установка Python-зависимостей
```bash
pip install clang
```

### Установка пути
Если путь libclang.cpp не повпадает с C:\Program Files\LLVM\bin\libclang.dll, нужно исправить путь в строке:
```python3
clang.cindex.Config.set_library_file("C:/Program Files/LLVM/bin/libclang.dll")
```
